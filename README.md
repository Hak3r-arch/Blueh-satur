
# ⚡ BLUEH-SATUR v4.0 ⚡

Saturador Bluetooth avanzado para auditorías de red en Termux.

---

### 🛠️ INSTALACIÓN PASO A PASO
Si te tira error de 'Module bluetooth not found' o plataforma no soportada, usá estos comandos:

1. **Dependencias del Sistema:**
```bash
pkg update && pkg upgrade -y
pkg install python git bluez bluez-utils libbluetooth-dev -y
```

2. **Librerías de Python:**
```bash
pip install rich
pip install pybluez2 --pre
```

3. **Ejecución:**
```bash
git clone https://github.com/Hak3r-arch/Blueh-satur.git
cd Blueh-satur
chmod +x arch-hacks
python arch-hacks
```
