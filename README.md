
# ⚡ BLUEH-SATUR v4.0 ⚡

Saturador Bluetooth para Termux.

### 🛠️ INSTALACIÓN SIN ERRORES
Para arreglar el error de 'No module named bluetooth', tirá estos comandos:

1. `pkg update && pkg upgrade -y`
2. `pkg install python git bluez libbluetooth-dev -y`
3. `pip install rich`
4. `pip install pybluez2 --pre`

### 🚀 EJECUCIÓN
```bash
git clone https://github.com/Hak3r-arch/Blueh-satur.git
cd Blueh-satur
chmod +x arch-hacks
python arch-hacks
```
