# ⚡ Tracker-blue ⚡
Script avanzado en Python para monitoreo y saturación Bluetooth.

## 🛠️ Requisitos
* **Termux** con Bluez (pkg install bluez)
* **Root** (Para usar la antena hci0)

## 🚀 Instalación y Uso
```bash
git clone https://github.com/Hak3r-arch/Tracker-blue
cd Tracker-blue
python saturador.py
```
